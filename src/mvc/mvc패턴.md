### 기능 정의서
1. 검색기 구현한다. (제목, 작성자, 내용)
2. 오라클과 연동하여 처리한다. (CRUD 작업)
3. 글쓰기를 구현한다. - insert - int
4. 글 삭제하기를 구현한다. - delete - int
5. 글 수정하기를 구현한다. - update - int
6. 전체조회를 구현한다. - select many - List<Map<>>
7. 상세보기를 구현한다. - select one - map
